Программа: Библиотека


Основные сущности
Book - книга (название, год издания, количество экземпляров всего и доступных для выдачи)
Author - автор (имя, фамилия)
Reader - читатель (имя, фамилия, email, телефон, блокировка)
Fine - штраф (сумма, причина, статус оплаты)

Функционал
Программа позволяет создавать, просматривать, изменять и удалять каждую из сущностей.
Читатель может взять и вернуть книгу, взятую книгу может продлить на дополнительное время.
Читатель может оплатить штраф.
Пользователь может просмотреть, кто опоздал со сдачей книги.
